📖 Cách lấy Page Access Token
Page Access Token cho phép ứng dụng đăng bài thay mặt fanpage của bạn thông qua Facebook Graph API.
1
Vào Facebook Developers
Truy cập developers.facebook.com → đăng nhập tài khoản Facebook của bạn.
2
Tạo App mới
Chọn My Apps → Create App. Chọn loại "Business" hoặc "None". Đặt tên app bất kỳ.
3
Thêm sản phẩm Pages API
Trong Dashboard app → Add Product → tìm và thêm "Facebook Login" hoặc "Pages API".
4
Lấy token qua Graph API Explorer
Vào Graph API Explorer.
Chọn app của bạn → nhấn "Generate Access Token".
Cấp quyền: pages_manage_posts, pages_read_engagement.
Chọn trang → copy Page Access Token.
5
Lấy Page ID
Vào trang fanpage → Giới thiệu → cuộn xuống tìm mục "ID trang". Hoặc vào Cài đặt trang → Thông tin trang.
6
Gia hạn token (quan trọng)
Token mặc định chỉ có hiệu lực 1-2 giờ. Để dùng lâu dài, dùng long-lived token:
Gọi API: GET /oauth/access_token?grant_type=fb_exchange_token&client_id={app_id}&client_secret={secret}&fb_exchange_token={token}
Token dài hạn có hiệu lực ~60 ngày.
💡 Lưu ý CORS: Khi gọi Graph API từ trình duyệt, Facebook cho phép CORS từ mọi origin nên tool này hoạt động bình thường mà không cần backend.
