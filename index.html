📖 Tạo Facebook App ID
Bạn cần App ID để xác thực với Facebook Login. Miễn phí, tạo trong 5 phút.
1
Vào Facebook Developers
Truy cập developers.facebook.com và đăng nhập bằng tài khoản Facebook của bạn.
2
Tạo App mới
Nhấn My Apps → Create App. Chọn loại "Business" hoặc "Consumer". Đặt tên app bất kỳ (ví dụ: "My Page Poster").
3
Thêm Facebook Login
Trong Dashboard → Add Product → Facebook Login → Web. Trong Site URL nhập http://localhost hoặc đường dẫn file HTML của bạn.
4
Bật quyền cần thiết
Vào App Review → Permissions, thêm: pages_manage_posts và pages_read_engagement. Ở chế độ Development, bạn dùng được ngay với tài khoản admin của app.
5
Lấy App ID
Quay về Dashboard, copy App ID (chuỗi số ở góc trên). Dán vào ô "Facebook App ID" trong trang đăng nhập của tool.
6
Thêm tài khoản tester (nếu cần)
Vào Roles → Testers, thêm tài khoản Facebook bạn muốn dùng với tool. Tài khoản admin của app không cần bước này.
⚠️ App Review: Để người khác ngoài admin/testers dùng tool này, app phải qua Facebook App Review. Với mục đích cá nhân (quản lý page của chính bạn), không cần App Review.
✅ Token được lưu trong localStorage trên máy tính của bạn. Không gửi đến bất kỳ server nào ngoài api.facebook.com.
